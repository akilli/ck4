# akilli/ckeditor4-build-inline

Custom CKEditor 4 inline build with following plugins

- [Basic Styles](https://ckeditor.com/cke4/addon/basicstyles)
- [Block Widget](https://ckeditor.com/cke4/addon/block)
- [Browser API](https://ckeditor.com/cke4/addon/browser)
- [Details Widget](https://ckeditor.com/cke4/addon/detail)
- [Enter Key](https://ckeditor.com/cke4/addon/enterkey)
- [Floating Space](https://ckeditor.com/cke4/addon/floatingspace)
- [Format](https://ckeditor.com/cke4/addon/format)
- [Gallery Widget](https://ckeditor.com/cke4/addon/gallery)
- [Grid Widget](https://ckeditor.com/cke4/addon/grid)
- [Link](https://ckeditor.com/cke4/addon/link)
- [List](https://ckeditor.com/cke4/addon/list)
- [Magic Line](https://ckeditor.com/cke4/addon/magicline)
- [Maximize](https://ckeditor.com/cke4/addon/maximize)
- [Media Widget](https://ckeditor.com/cke4/addon/media)
- [Quote Widget](https://ckeditor.com/cke4/addon/quote)
- [Section Widget](https://ckeditor.com/cke4/addon/section)
- [Table](https://ckeditor.com/cke4/addon/table)
- [Undo](https://ckeditor.com/cke4/addon/undo)
